# 抖音达人背调助手

抖音达人背调与筛选助手。输入博主主页链接或视频分享链接，一键获取博主全维度画像。

## MCP 服务配置

本技能依赖以下 MCP 服务，运行时**必须**先读取 `mcp-config.json` 获取服务 URL：

| 环境变量名 | MCP 服务 | mcpId | 用途 |
|-----------|----------|-------|------|
| `$MCP_DOUYIN_BLOGGER_URL` | 获取抖音博主数据 | 1077 | 获取博主认证、粉丝、获赞、作品等全维度数据 |

### 配置读取方式

**约束：调用方 agent 必须在执行任何 MCP 调用前，先读取 mcp-config.json 文件**

```python
import json

with open("mcp-config.json", "r", encoding="utf-8") as f:
    config = json.load(f)

douyin_blogger_url = config["MCP_DOUYIN_BLOGGER_URL"]["url"]
```

## 工作流程

### 第一步：提取博主标识

从用户输入中提取博主信息，常见输入格式：
- 博主主页链接：`https://www.douyin.com/user/...`
- 视频分享链接：`https://v.douyin.com/...`（从视频反查博主）
- 分享口令：包含链接的文字段落
- 抖音号：直接输入抖音号

### 第二步：调用博主数据接口

1. **读取配置**：首先读取 `mcp-config.json` 获取 MCP 服务的 URL
2. **验证配置**：检查 URL 是否已配置
3. **调用获取**：

```bash
# 先发现可用工具
python3 scripts/call_mcp.py list "$MCP_DOUYIN_BLOGGER_URL"

# 获取博主数据
python3 scripts/call_mcp.py call "$MCP_DOUYIN_BLOGGER_URL" <发现的工具名> \
  --params '{"blogger_url": "<博主主页链接或视频链接>"}'
```

### 第三步：输出背调报告

## 输出格式

```
## 抖音达人背调报告：<博主昵称>

### 身份信息
- 昵称：
- 抖音号：
- 认证信息：<是否认证 / 认证类型>
- IP属地：

### 数据概览
| 指标 | 数值 |
|------|------|
| 粉丝数 | X |
| 获赞数 | X |
| 作品数 | X |

### 背调评估
- 认证状态：<已认证/未认证>
- 粉丝量级：<头部/腰部/尾部>
- 互动率参考：<获赞数/粉丝数 比值>
- 活跃度：<基于作品数和更新频率>

### 合作建议
<基于以上数据的简要评估>
```

## 注意事项

- 从用户输入中自动提取链接或抖音号，无需用户手动格式化
- 如果用户给的是视频链接而非博主主页，需先获取视频数据再反查博主
- 数字做可读化处理（如 1234567 → 123.5万）
- 粉丝量级参考：100万+ 头部，10-100万 腰部，10万以下 尾部
- 如果接口返回失败，提示用户检查链接是否有效
